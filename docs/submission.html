What we built

A building-operator dashboard that pulls live work orders from the CriticalAsset GraphQL API for 240 2nd Avenue, joins them to linked assets and locations, and turns facility data into a Monday-morning decision surface.

  • ✓ OAuth2 / session auth via server-side scripts (no browser secrets)
  • ✓ Live pull: 3 work orders, 100 assets, full location tree
  • ✓ Counter row: Open · In Progress · Overdue
  • ✓ Filterable table with asset, category, location, due date
  • ✓ Detail panel with full work order + linked assets
  • ✓ Bonus: category grouping, building stats, student signal structurer
