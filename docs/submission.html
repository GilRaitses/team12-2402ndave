Why API keys are not in GitHub

  • GitHub Pages is public static hosting. Any key in HTML/JS is visible to everyone via View Source.
  • Git history is permanent. Even deleted secrets remain in old commits.
  • GitHub Secrets only inject into Actions/Codespaces at build time — they do not protect a static site served to browsers.
  • Correct pattern: store OPENAI_API_KEY and CA_CLIENT_SECRET in local .env (gitignored). Python api_server.py reads them server-side only.
  • Setup: python3 scripts/setup_env.py — interactive local paste, never committed.
