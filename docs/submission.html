Infrastructure note

No AWS S3, SQS, or custom cloud infra required. Stack: Python hydration scripts → JSON cache → GitHub Pages static UI → optional backend/proxy.py for local dev.

Amazon Quick / Kiro listed on setup page as optional vibe-coding tools — we shipped custom code instead (allowed per rules).

3-minute demo script

Minute 1 — Signal

Students at 240 2nd Avenue feel facility issues before the system does. Our product captures that signal and connects it to real work orders and public evidence.

Minute 2 — Product

Dashboard: 4 live WOs, filter HVAC, open steam inspection. Workflow: enter bathroom signal → AI structures location/issue → 311/HPD context → compliance obligations → recommended dispatch.

Minute 3 — Decision + verification

Record to CriticalAsset via script. Closure buttons ask: fixed / still / worse. Limitations: prototype, human review required, not legal advice.
